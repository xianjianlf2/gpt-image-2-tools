AI 图生图：拖一张图进来就能改

AI Image-to-Image: drop a picture and edit it

上传或拖入一张图，用一句话描述你想要的改动，直接生成新版本。

Drop in an image, describe the change you want in words, and generate a new version.

开始创作 →

图生图（Image-to-Image）是在已有图像基础上做编辑、改风格或扩展。GPT-Image-2 Tools 把它和文生图合并在同一个输入框：拖图进来就自动切到改图模式，配合 Prompt 即可。纯客户端、密钥只存本机，适合改风格、补细节、出多版方案。

Image-to-image edits, restyles or extends an existing picture. GPT-Image-2 Tools merges it with text-to-image in one input: drop an image and it switches to edit mode automatically, then describe what to change. Fully client-side, key stays on your device — ideal for restyling, refining details and producing variations.

怎么用 · How it works

  1. 1 填好 API 地址和密钥（只存浏览器） Add your API base URL and key (stored only in your browser)
  2. 2 把图片拖进输入框，自动进入改图模式 Drag an image into the input — it switches to edit mode
  3. 3 写下想要的改动，生成；可对结果继续迭代 Describe the change and generate; keep iterating on results

常见问题 · FAQ

图生图和文生图怎么切换？

不用切换。输入框是统一的——没图就是文生图，拖入图片就自动变成图生图/改图。

How do I switch between modes? You do not. The input is unified — no image means text-to-image; drop an image and it becomes image-to-image automatically.

可以一次改多张图吗？

可以附多张参考图，并设置一次生成多张结果，方便对比挑选。

Can I edit multiple images? You can attach multiple reference images and generate several outputs at once to compare.

改完的图能继续改吗？

可以。任意结果点「以此继续」，把它作为新输入接着改，形成迭代链路。

Can I keep editing the result? Yes. Click "continue from this" on any result to feed it back as the new input and iterate.
