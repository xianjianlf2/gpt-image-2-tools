文生图（Text-to-Image）就是用自然语言描述生成图像。GPT-Image-2 Tools 接任意 OpenAI 兼容图像端点，纯客户端运行——你填自己的 API 地址和密钥，浏览器直连，无服务器中转，密钥只存本机。适合做插画、封面、概念图、海报草稿。

Text-to-image turns a natural-language prompt into a picture. GPT-Image-2 Tools works with any OpenAI-compatible image endpoint and runs fully client-side — you bring your own API base URL and key, the browser connects directly with no server relay, and the key stays on your device. Great for illustrations, covers, concept art and poster drafts.

怎么用 · How it works

  1. 1 在设置里填好 API 地址和密钥（只存浏览器） Add your API base URL and key in Settings (stored only in your browser)
  2. 2 在输入框写下画面描述，选尺寸/质量/张数 Type your prompt, pick size / quality / count
  3. 3 回车生成，满意的结果可一键「以此继续」迭代 Press Enter to generate, then iterate on any result in one click

常见问题 · FAQ

文生图需要注册吗？

不需要注册本站。它是纯客户端工具，你用自己的 OpenAI 兼容密钥，配置只存在浏览器本地。

Do I need to sign up? No signup here. It is a fully client-side tool — you use your own OpenAI-compatible key and your config is stored only in your browser.

我的密钥安全吗？

密钥只存在你浏览器的 localStorage，浏览器直连你填的端点，绝不经过我们的服务器。

Is my key safe? Your key lives only in your browser localStorage and connects directly to the endpoint you set — it never passes through our servers.

支持哪些模型？

任何 OpenAI 兼容的图像生成端点，模型名可在设置里搜索选择或手动填写。

Which models are supported? Any OpenAI-compatible image endpoint; pick the model from the searchable list in Settings or type it in.
