关于 GPT-Image-2 Tools

GPT-Image-2 Tools 是一个面向 OpenAI 兼容图像接口的纯客户端创作工具。一个输入框搞定一切：直接写 Prompt 就是文生图，拖一张图进来自动变成改图 / 局部重绘。可调尺寸、质量、张数，结果可一键继续迭代。 无需注册、无服务器中转——你的密钥只存在浏览器本地，永不离开本机。

GPT-Image-2 Tools is a fully client-side image playground for any OpenAI-compatible image endpoint. One input does it all: write a prompt to generate, or drop an image to edit / inpaint. Tune size, quality and count, then iterate on any result in one click. No signup, no server relay — your key stays in your browser and never leaves your device.
